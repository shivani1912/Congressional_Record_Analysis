# Congressional_Record_Analysis
Analysis of Congressional data to identify partisanship and themes related to each party. 
Data Background: 
This data originally appeared in Gentzkow and Shapiro (GS; 2010) and considers text of the 2005 Congressional Record, containing all speeches in that year for members of the United States House and Senate. In particular, GS record the number times each of 529 legislators used terms in a list of 1000 phrases (i.e., each document is a year of transcripts for a single speaker).
